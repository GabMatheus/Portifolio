Meus Projetos

Por trabalhar em uma empresa privada, grande parte dos projetos que desenvolvi é destinada exclusivamente para o uso interno, não podendo ser divulgada publicamente.

No entanto, posso compartilhar algumas pequenas automações que criei para melhorar atividades rotineiras, as quais estão disponíveis no meu Github:

Por exemplo, este projeto de automação que desenvolvi para gerar códigos de barras para produtos. Esse projetinho foi criado porque alguns produtos têm códigos de barras ilegíveis ou de difícil acesso para os funcionários. Assim, eles podem gerar e manter uma cópia próxima para facilitar a entrega de informações aos clientes.

Outro exemplo é este projeto, que desenvolvi para consultar CNPJs na API da Receita Federal e retornar alguns dados. Este projeto foi criado para extrair informações necessárias sobre empresas e gerar um arquivo TXT no final. Esse arquivo é então importado no ERP Dominio Sistemas, facilitando o cadastro automático de clientes e fornecedores na área de contabilidade, através de uma funcionalidade do sistema que utilizam.

Outro projeto que fiz é este, que mostra a distância entre cidades. O app consulta a API do Google Maps e retorna a distância entre cidades, diretamente ou de uma para todas as outras registradas na API, atualmente limitado ao Brasil.

Por fim, este projeto renomeia arquivos PDF. Os arquivos de comprovantes da empresa vinham com nomes estranhos. Com esse script em Python, conseguimos extrair o CNPJ da empresa e renomear o arquivo principal de forma que facilite a consulta, sem a necessidade de abrir cada arquivo para visualizar o conteúdo.
